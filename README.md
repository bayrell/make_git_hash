# Make git hash

Show help:
```
./make_git_hash.py --help
```

Show git hash
```
./make_git_hash.py
```

Show git hash by prefix
```
./make_git_hash.py --prefix prefix
```

Select hash:
```
./make_git_hash.py --select number
```

Select hash and apply:
```
./make_git_hash.py --select number --apply
```
